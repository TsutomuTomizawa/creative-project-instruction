# LP制作 - 詳細インストラクション

## A. ランディングページ制作

### 📚 使用ナレッジベース

#### LP制作の参照ファイル
```yaml
# モジュール型LP制作フロー
リサーチ・設計フェーズ:
  - /knowledge-base/lp/01-research.md             # リサーチフェーズ総合ガイド
  - /knowledge-base/lp/02-module-selection.md     # モジュール選択ガイド
  - /knowledge-base/lp/03-story-modules.md        # ストーリーモジュール集

制作・実装フェーズ:
  - /knowledge-base/lp/04-components.md           # コンポーネント仕様ガイド
  - /knowledge-base/lp/05-copywriting.md          # コピーライティングガイド
  - /knowledge-base/lp/06-optimization.md         # 最適化ガイド

# ワークフロー:
# Phase 0 (リサーチ) → Phase 0.5 (モジュール選択) → Phase 1 (ストーリーフロー構築) → Phase 2 (セクション制作)
```

## 🔍 準備フェーズ: 初期評価

### 軽量な初期チェック

制作を開始する前に、最小限の準備を行います：

1. **基本要件の確認**
   - ユーザー提供情報の把握
   - 明らかに不足している情報の特定
   - 基本的なプラットフォーム要件の確認

2. **段階的参照計画**
   - Phase 0: 基本的なガイドラインのみ参照
   - Phase 0.5: モジュール選択の基準を確認
   - Phase 1: ストーリーフローの構築
   - Phase 2: 各セクションで詳細なコンポーネント情報を参照

```

- 前セクション終了部: 「しかし、これだけでは十分ではありません...」
- 当セクション開始部: 「そこで重要になるのが〜です」

□ **発展型接続**
- 前セクション終了部: 「基本的な価値は理解いただけたと思います」
- 当セクション開始部: 「では、なぜ多くの企業が選ぶのか、その理由を...」

□ **疑問解消型接続**
- 前セクション終了部: 「でも、本当に自分でもできるのか不安ですよね」
- 当セクション開始部: 「ご安心ください。充実したサポートがあります」
```
