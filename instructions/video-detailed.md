## 🔍 準備フェーズ: 初期評価

### 軽量な初期チェック

動画広告制作を開始する前に、最小限の準備を行います：

1. **基本要件の確認**
   - ユーザー提供情報の把握
   - 動画の尺・用途・配信先の確認
   - 明らかに不足している情報の特定

2. **段階的参照計画**
   - Phase 0: 基本的なガイドラインのみ参照
   - Phase 1: ストーリーフローの構築、パターン参照
   - Phase 2: 各シーンで詳細なコンポーネント情報を参照

```

━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**実行手順：**

1. **商品/サービス理解**
   - WebSearchで「[商品・サービス名] 特徴 メリット」を検索
   - 参考URLがある場合はWebFetchで詳細情報を取得
   - 公式サイトから提供価値とブランディング要素を抽出

2. **配信媒体分析**
   - WebSearchで「[プラットフォーム名] 動画広告 トレンド 2025」を検索
   - 各プラットフォームの最新ガイドライン確認
   - 効果的な動画形式と審査基準の把握

3. **競合・市場分析**
   - WebSearchで「[商品・サービス名] 競合 比較」を検索
   - YouTubeで「[商品・サービス名]」を検索し、関連動画を分析
   - 業界の成功事例と表現手法の調査

4. **ターゲット行動分析**
   - WebSearchで「[ターゲット層] 動画視聴 傾向」を検索
   - プラットフォーム別の視聴行動パターンを調査

**リサーチ結果の提示：**
収集した情報を以下の形式で整理して提示：
- 商品理解サマリー
- 競合動画の特徴
- ターゲット層の推定プロファイル
- 推奨される動画構成の方向性

🎭 **ナレーションテイストの選択**

どのナレーションテイストを選択しますか？

1. プロフェッショナル型（信頼性重視）
   - 客観的データと事実ベース
   - 権威的で説得力のある語り
   - B2B、高額商材向け

2. フレンドリー型（体験談型）★おすすめ
   - 親しみやすい友達口調
   - 個人的な体験と感想を中心
   - SNS広告、若年層向け

3. エネルギッシュ型（活力重視）
   - テンポが速く、感嘆符多用
   - 興奮と期待感を演出
   - スポーツ・エンタメ商材向け

4. エモーショナル型（感情訴求）
   - 感情に強く訴える
   - ストーリー性重視
   - ブランディング広告向け

5. ニュートラル型（バランス型）
   - 幅広いターゲットに対応
   - 標準的な語り口
   - 汎用性が高い

━━━━━━━━━━━━━━━━━━━━━━━━━━

💭 **体験要素の組み込み（複数選択可）**

どの体験要素を組み込みますか？

□ 個人的な悩み・課題
  例：「夕方は特に目がぼやけて...」

□ 使用前の失敗談
  例：「今まで飲んでいたサプリじゃ物足りなくて」

□ 使用開始のきっかけ
