## 🔍 準備フェーズ: 初期評価

### 軽量な初期チェック

動画広告制作を開始する前に、最小限の準備を行います：

1. **基本要件の確認**
   - ユーザー提供情報の把握
   - 動画の尺・用途・配信先の確認
   - 明らかに不足している情報の特定

2. **段階的参照計画**
   - Phase 0: 基本的なガイドラインのみ参照
   - Phase 1: ストーリーフローの構築、パターン参照
   - Phase 2: 各シーンで詳細なコンポーネント情報を参照

## 📊 成果物標準

```
納品物:
1. フックパターン一覧（初期5パターン以上）
2. 選定訴求型のバリエーション展開（3-5パターン）
3. 各バリエーションのナレーション原稿（タイムコード付き）
4. バリエーション別シーン構成表（ナレーション・ビジュアル・キャプション・音響）
5. 制作指示書（撮影・編集指示、バリエーション別）
6. AI素材生成用JSON Context Profile（AI使用時、各バリエーション対応）
7. プラットフォーム別最適化案（必要に応じて）
8. A/Bテスト推奨案（バリエーション間の比較ポイント）
```

# CloudCRM動画広告制作指示書

## プロジェクト概要
**選定した訴求型**: 恐怖訴求型
**理由**: ターゲット層の調査から、現状の非効率な管理に強い不満を持っていることが判明。失敗リスクを強調することで行動喚起を促す。

## バリエーションA：業界全体のリスク訴求

### ナレーション原稿（30秒）
| 時間 | ナレーション | 文字数 |
|------|-------------|--------|
| 0:00-0:05 | エクセル管理で年間20時間もムダにしていませんか？ | 23文字 |
| 0:05-0:10 | 競合他社はすでにシステム化で効率を2倍に。 | 20文字 |
| 0:10-0:20 | CloudCRMなら自動化により作業時間を80%削減、リアルタイムで情報共有。 | 35文字 |
| 0:20-0:25 | 大手500社が導入済み、満足度98%。 | 18文字 |
| 0:25-0:30 | 今すぐ14日間無料でお試しください。 | 18文字 |

## シーン1（0-5秒）：問題提起

### 必要な素材

2. 散らかったデスク環境（書類の山、複数のモニター）
3. 時計のグラフィック（時間が無駄に流れる表現）
