## 🔍 準備フェーズ: 初期評価

### 軽量な初期チェック

動画広告制作を開始する前に、最小限の準備を行います：

1. **基本要件の確認**
   - ユーザー提供情報の把握
   - 動画の尺・用途・配信先の確認
   - 明らかに不足している情報の特定

2. **段階的参照計画**
   - Phase 0: 基本的な動画要件のみ確認
   - Phase 1: 該当する尺の成功パターン概要を参照
   - Phase 2: 各シーンで具体的な演出例を参照

Phase 0の情報を基に、視聴者の心理的な流れを設計します。

🎭 **視聴者の心理的な流れ**（[尺]秒版）

1️⃣ **フック段階**（0-3秒）
   状態: スクロール中/他の動画視聴後
   心理: 「何これ？」「自分に関係ある？」
   必要な要素: 強烈なビジュアル、共感を呼ぶ問いかけ

2️⃣ **問題提起段階**（3-8秒）
   状態: 動画に注目し始める
   心理: 「確かにそれ困ってる」「もっと知りたい」
   必要な要素: 具体的な課題、感情的な共感

3️⃣ **解決提示段階**（8-20秒）
   状態: 解決策を理解する
   心理: 「なるほど」「これなら解決できそう」
   必要な要素: 明確な解決策、信頼できる根拠

4️⃣ **実証段階**（20-25秒）※30秒以上の場合
   状態: 信頼性を確認
   心理: 「本当に効果あるの？」「他の人も使ってる？」
   必要な要素: 実績、testimonial、デモ

5️⃣ **CTA段階**（最後5秒）
   状態: 行動を促される
   心理: 「試してみたい」「どうすればいい？」
   必要な要素: 明確な次のステップ、簡単さの強調

📊 **音声OFF対策**
- 全シーンに字幕/テロップを配置
- ビジュアルだけで理解できる構成
- 重要メッセージは画面中央に配置

### 各シーン制作の流れ

同様に以下のシーンを順次制作：

```
制作順序（30秒動画の例）:
1️⃣ オープニング（フック）：0-3秒
2️⃣ 問題提起：3-8秒
3️⃣ 解決提示：8-18秒
4️⃣ 実証/信頼性：18-25秒
5️⃣ CTA（行動喚起）：25-30秒

各シーン制作時の5要素：
1. ターゲット視聴者の状態
2. メッセージレベルの訴求概要
3. 詳細な訴求内容
4. コンテンツ要素（テキスト、ビジュアル、オーディオ、キャプション）
5. プラットフォーム最適化

必ず1シーンずつ確認を取ってから次に進む
```
