# pricing-comparison: プラン比較表

## 推奨用途
- 複数プランの比較
- 機能差の明確化
- 推奨プランの誘導

## 表示形式
1. **カード型**: 各プランをカード形式で並列表示
2. **テーブル型**: 機能比較表
3. **スライダー型**: モバイル対応

## 必須要素
```
□ プラン名
□ 価格（月額/年額）
□ 主要機能リスト
□ 推奨バッジ（人気プラン）
□ CTAボタン
□ 制限事項の明記
```

## 心理的効果の活用
- **アンカリング効果**: 最高額プランを左端に配置
- **デコイ効果**: 中間プランを推奨として強調
- **損失回避**: 「今だけ」「限定」要素

## 業界別推奨構成での使用例
**SaaS/B2B（標準7セクション）**:
```
1. hero-split
2. social-logos（信頼性）
3. value-process（仕組み説明）
4. value-3column（主要機能）
5. social-testimonials（事例）
6. pricing-comparison（料金） ← このコンポーネント
7. cta-primary + faq-accordion
```
