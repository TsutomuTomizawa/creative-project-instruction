# edu-guarantee: 成果保証型

## 適用条件
- 受講料5-50万円
- 明確な成果定義可能
- 返金保証可能

## 成功の型
```
1. ヒーロー: 成果保証の約束
2. 実績: 受講生のビフォーアフター
3. カリキュラム: 段階別の詳細
4. 講師: 実績と経歴
5. サポート: 個別フォロー体制
6. 証言: 動画インタビュー
7. 保証: 条件と返金規定
8. CTA: 無料説明会予約
```

## 信頼構築の要素
```
実績の可視化:
- 合格率/就職率の推移グラフ
- 受講生の作品/成果物
- メディア掲載/受賞歴

サポートの充実:
- 質問対応時間
- 個別面談回数
- 卒業後のフォロー
```

## 説明会への誘導
```
LP → 説明会予約（5-10%）
  → 説明会参加（70%）
  → 申込み（30-50%）
```

## 推奨コンポーネント構成
```
1. hero-video（講師メッセージ）
2. value-beforeafter（成果）
3. value-process（カリキュラム）
4. social-testimonials（受講生の声）
5. pricing-single（受講料）
6. guarantee-badge（返金保証）
7. faq-accordion（よくある質問）
8. cta-primary（申込み）
```
