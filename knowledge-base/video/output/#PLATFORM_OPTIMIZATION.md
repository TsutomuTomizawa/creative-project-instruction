# OUTPUT_CONTEXT: プラットフォーム最適化 - 各配信先に特化した動画仕様と戦略

## 概要
主要動画プラットフォームの技術仕様、視聴者行動、アルゴリズムを考慮し、各プラットフォームで最大の効果を発揮する動画制作と配信戦略のガイドです。

## プラットフォーム別詳細仕様

### YouTube
**技術仕様**:
```json
{
  "video_specs": {
    "aspect_ratio": ["16:9", "9:16（Shorts）"],
    "resolution": {
      "minimum": "1280x720 (HD)",
      "recommended": "1920x1080 (Full HD)",
      "maximum": "3840x2160 (4K)"
    },
    "frame_rate": ["24fps", "25fps", "30fps", "60fps"],
    "bitrate": {
      "1080p_30fps": "8 Mbps",
      "1080p_60fps": "12 Mbps",
      "4K_30fps": "35-45 Mbps"
    },
    "file_format": "MP4 (H.264/H.265)",
    "max_file_size": "256GB",
    "max_duration": "12時間"
  }
}
```

**視聴者行動特性**:
- 平均視聴時間: 11分
- 音声ON率: 95%（PC）、70%（モバイル）
- 5秒後のスキップ率: 30%
- 最適な動画長: 10-15分（通常）、60秒以内（Shorts）

**アルゴリズム最適化**:
```
重要指標:
1. 視聴時間（Watch Time）- 最重要
2. クリック率（CTR）- サムネイル効果
3. 視聴者保持率 - 特に最初の15秒
4. エンゲージメント - いいね、コメント、シェア
5. セッション時間 - 関連動画への誘導
```

**広告フォーマット別戦略**:
| フォーマット | 長さ | 特徴 | 最適な用途 |
|------------|------|------|----------|
| TrueView インストリーム | 制限なし | 5秒後スキップ可 | ブランド認知、詳細説明 |
| バンパー広告 | 6秒 | スキップ不可 | リマインダー、印象付け |
| TrueView ディスカバリー | - | 検索結果に表示 | 能動的な視聴者向け |

```json

  "video_specs": {

      "aspect_ratio": ["1:1", "4:5", "16:9"],
      "resolution": "1080x1080 推奨",

    },
    "reels": {
      "aspect_ratio": "9:16",
      "resolution": "1080x1920",
      "duration": "最大90秒"
    }
  }
}
```

**視聴者行動特性**:
- 無音視聴率: 85%
- 平均視聴時間: 3秒（フィード）
- スクロール速度: 1投稿あたり1.7秒
- 最適な動画長: 15-30秒

**コンテンツ最適化**:
```
フィード投稿:
- 最初の3秒で価値提示必須
- 字幕は画面の80%サイズ
- コントラスト高めの色使い
- 正方形or縦型推奨

ストーリーズ:
- フルスクリーン活用
- インタラクティブ要素追加
- 上下20%は操作エリア考慮
- 15秒単位で構成
```

## マルチプラットフォーム展開戦略

### アスペクト比変換マトリックス
```
元素材: 16:9（マスター版）

変換パターン:
16:9 → 1:1（Instagram）
- 方法: 上下をクロップ
- 注意: 重要要素は中央配置

16:9 → 9:16（TikTok/Reels）
- 方法: ズーム＆センタークロップ
- 注意: 別撮り推奨

16:9 → 4:5（Instagram）
- 方法: 左右を少しクロップ
- 注意: 字幕位置の調整
```

### セーフゾーン設計
```
全プラットフォーム対応レイアウト:

│  10%（装飾のみ）    │

│ │                 │ │
│ └─────────────────┘ │
│  10%（装飾のみ）    │
└─────────────────────┘
```

### 曜日別戦略
```
月曜：モチベーション系コンテンツ
火-木：教育・情報系コンテンツ
金曜：エンタメ・軽めのコンテンツ
週末：
- B2C: 高エンゲージメント期待
- B2B: 低活性、避ける
```
