```

### アスペクト比変換戦略
```
元素材: 9:16縦型（マスター版）

変換パターン:
9:16 → 1:1（Instagram フィード）
- 方法: 上下を適切にトリミング
- 注意: 重要要素は中央60%に配置

9:16 → 16:9（YouTube通常）
- 方法: 左右に背景追加またはぼかし
- 注意: 別素材での制作も検討

9:16 → 4:5（Instagram フィード）
- 方法: 上下を少しトリミング
- 注意: テキストの再配置
```

```
